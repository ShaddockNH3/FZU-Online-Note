# FZU Online Note

一个轻量级在线笔记 Web 应用，选题方向为“在线笔记”。项目使用原生 HTML、CSS 和 JavaScript 实现，不依赖后端服务，数据通过浏览器 `localStorage` 持久化保存。

## 项目基本信息

- 项目名称：FZU Online Note
- 选题方向：在线笔记
- 技术路线：HTML + CSS + JavaScript + localStorage

## 运行方式

本项目为纯前端静态页面，无需安装依赖，也不需要配置数据库。

方式一：直接运行

1. 下载或克隆本仓库。
2. 双击打开根目录下的 `index.html`。
3. 在浏览器中即可使用在线笔记功能。

方式二：本地静态服务运行

```bash
python -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173/index.html
```

## 功能说明

- 新增笔记：填写标题、内容、标签、主题、状态后保存。
- 查看笔记：页面会动态渲染笔记卡片列表。
- 编辑笔记：点击“编辑”后回填表单并更新原笔记。
- 删除笔记：点击“删除”后确认移除。
- 复制笔记：快速复制已有笔记生成副本。
- 置顶笔记：重要笔记可置顶展示。
- 搜索笔记：支持按标题、内容、标签和角色主题搜索。
- 筛选笔记：支持全部、置顶、今日新增三种筛选。
- 排序笔记：支持按最近更新、创建时间、标题排序。
- 数据统计：展示总笔记数、置顶数、标签数、今日新增数。
- 数据持久化：笔记数据保存在浏览器 `localStorage` 中，刷新页面后仍会保留。

## 技术选型

- HTML：构建页面结构和表单内容。
- CSS：实现响应式布局、视觉风格和交互反馈。
- JavaScript：实现表单校验、DOM 渲染、事件处理和 CRUD 逻辑。
- localStorage：作为轻量级本地数据存储方案。

选择原生前端技术的原因是项目规模较轻，符合课程基础版要求，也便于直接打开和提交，不需要额外安装框架或配置运行环境。

## 素材来源说明

角色主题参考萌娘百科《巧克甜恋》相关页面，当前页面包含雪村千绘莉、天宫美久栗、御园莓华、舞羽娜娜、百濑辉夜、百濑美月和小花。

角色图片默认使用萌娘百科图片直链。为了保证答辩时即使网络不稳定也能展示，项目在 `assets/local-fallback/` 保留了一份本地原图缓存作为回退，但该目录已写入 `.gitignore`，不会进入 Git 仓库。

## 数据与数据库说明

本项目未使用 PHP、MySQL 或其他后端数据库，因此不需要导入 `.sql` 文件，也不需要配置数据库连接。

所有笔记数据均保存在当前浏览器的 `localStorage` 中。更换浏览器、清除浏览器数据或使用无痕模式时，原有笔记数据可能无法保留。
